The Parity Game

Increase Difficulty        

 


 


 


 


 


 

Instructions:

  1. The "Data Input Person" randomly select any of the boxes by tapping or clicking on them.
  2. The "Parity Checker" will make things more complicated by clicking on the "Increase Difficulty" checkbox.
  3. The "Data Input Person" changes one of the boxes without the "Parity Checker" looking.
  4. By using even parity the "Parity Checker" will be able to determine which box was changed.

Based on the Error Detection Activity from CS Unplugged http://csunplugged.org/error-detection.
